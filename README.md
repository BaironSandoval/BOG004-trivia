## 1. Resumen del proyecto

Esta es un juego de trivia donde se evalua el conocimiento acerca de paises y capitales. Dandole al usuario opciones de respuestas.

## 2. Como se desarrollo

iniciamos diseñando el prototipo final atravez de una herramienta de diseño, en la cual nos basamos para construir la estructura visual y funcional del juego
